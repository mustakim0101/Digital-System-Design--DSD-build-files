# ✅ Common for all instructions

D7–D5 = Opcode (goes to your 74LS138)


# 1) Opcode = `000`  → ALU / R-type

Purpose: do one of your ALU operations (16 ops)

* **ALU selector (funct) = D4–D1**  ✅ this goes to S3..S0 (when ALU=1)
* **D0 = don’t care** (you can keep 0 or 1)

ALU_Control_3 (Most Significant Bit):
    *   It is '1' only when `ALUOp=10` AND `Func3=1`.
    *   Equation: `ALU_Control_3 = (ALUOp1 AND NOT(ALUOp0) AND Func3)`

ALU_Control_2 :
    *   Equation: `ALU_Control_2 = (ALUOp1 AND NOT(ALUOp0) AND Func2)`

ALU_Control_1 :
    *   Equation: `ALU_Control_1 = (ALUOp1 AND NOT(ALUOp0) AND Func1)`

ALU_Control_0 (Least Significant Bit):
    *   It is '1' in two cases:
        1.  When `ALUOp=01` (for SUBTRACT).
        2.  When `ALUOp=10` AND `Func0=1` (for R-Type).
    *   Equation: `ALU_Control_0 = (NOT(ALUOp1) AND ALUOp0) OR (ALUOp1 AND NOT(ALUOp0) AND Func0)`


✅ In this format:

* **There is NO rs/rt/rd encoded.**
* Inputs come from **fixed or mux-selected registers** (your RA/RB forced logic).

So for opcode `000`:

* **ALU selector = D4 D3 D2 D1**
* **rs/rt/rd = not present**
* **offset = not present**



----------------------------------------------------------------------------------------------------------------
# 2) Opcode = `001`  → **LW (load word)**

**Purpose:** address = Reg[rs] + offset, then load memory data

* **rs (base register) = D4–D3**
* **offset (signed 3-bit) = D2–D0**  ✅ this goes into sign extender
* **rt is not encoded separately**
* **destination (write register) = RT bits = D2–D1** (as you requested for WA)

So for opcode `001`:

* **rs = D4 D3**
* **offset = D2 D1 D0**
* **write destination WA = D2 D1 (RT field you’re using)**
* **ALU selector is forced to ADD (0000)** (because LW)

----------------------------------------------------------------------------------------------------------------
# 3) Opcode = `010`  → **SW (store word)**

**Purpose:** address = Reg[rs] + offset, then store data to memory

* **rs (base register) = D4–D3**
* **offset (signed 3-bit) = D2–D0**
* no register write

So for opcode `010`:

* **rs = D4 D3**
* **offset = D2 D1 D0**
* **WA unused (RegWrite=0)**
* **ALU selector forced to ADD (0000)**

----------------------------------------------------------------------------------------------------------------
# 4) Opcode = `011`  → **BEQ (branch equal)**

**Purpose:** compare two registers (Reg[rs] vs Reg[rt]) using SUB

* **rs = D4–D3**
* **rt = D2–D1**
* **D0 don’t care**

So for opcode `011`:

* **rs = D4 D3**
* **rt = D2 D1**
* **ALU selector forced to SUB (0001)**
