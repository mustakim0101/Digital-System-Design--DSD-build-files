# ✅ Final Address-Select Wiring (RA0/RA1, RB0/RB1, WA0/WA1)

## Signals used

From your opcode decoder (74LS138 outputs active-low):

* **ALU = ¬Y0’**  (opcode = 000)
* **LW  = ¬Y1’**  (opcode = 001)
* **SW  = ¬Y2’**  (opcode = 010)
* **BEQ = ¬Y3’**  (opcode = 011)

From Instruction Register outputs:

* **IR4, IR3 = RS bits**
* **IR2, IR1 = RT bits**
* **IR2..IR0 = offset (lw/sw)**

All muxes we use are **74LS157**.

# 1) RA (Read Port A Address) — sometimes fixed, sometimes RS

### Goal:

* If **ALU=1** → RA = `00` (fixed R0 for ALU-type)
* Else → RA = `RS = IR4..IR3` (lw/sw/beq use base register = RS)

## Use one 74LS157 (call it U_RA)

### Power / enable

* **pin16 → +5V**
* **pin8  → GND**
* **pin15 (/G) → GND** (always enabled)

### Select

* **pin1 (SEL) → ALU**

### Channel wiring

#### RA0 (bit0) = output of channel 1

* **pin2 (A1) = IR3**   (when ALU=0 → use RS bit0)
* **pin3 (B1) = GND**   (when ALU=1 → fixed 0)
* **pin4 (Y1) = RA0**

#### RA1 (bit1) = output of channel 2

* **pin5 (A2) = IR4**   (when ALU=0 → use RS bit1)
* **pin6 (B2) = GND**   (when ALU=1 → fixed 0)
* **pin7 (Y2) = RA1**

### Connect RA outputs to ReadPortA mux bank (all 74LS153)

For every 74LS153 in **ReadPortA**:

* **RA0 → pin14 (A)**
* **RA1 → pin2  (B)**

✅ Behavior:

* ALU=1 → RA = 00 (R0)
* ALU=0 → RA = IR4..IR3 (RS)
