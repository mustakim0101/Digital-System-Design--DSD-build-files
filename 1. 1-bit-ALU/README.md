# 1-Bit ALU (Optimized Build)

## Overview
This folder contains the optimized 1-bit ALU design version.

Compared to the prototype design, this version is more practical in build complexity:
- uses fewer ICs
- uses fewer wires
- needs less power and GND distribution effort
- takes less breadboard space

## Suggested Reading Path
1. Read this file for the optimized version summary.
2. Review the earlier concept in [Prototype/README.md](./Prototype/README.md).
